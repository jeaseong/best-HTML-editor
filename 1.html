atom

Atom is an HMTL editor that came out in 2014 and gained tremendous momentum since.

Atom is a free, open-source code editor, and it was developed by the GitHub team.

Atom uses a free software license for its package and it is maintained by the GitHub community.

It aims to offer a premium feel to the editor while keeping it completely free. As well as the flexibility to customize the software itself.
